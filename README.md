# Ajuste de Distribuciones para el RCS

Repositorio enfocado en el análisis estadístico y ajuste probabilístico del Requerimiento de Capital de Solvencia (RCS) mediante distintas distribuciones de probabilidad utilizando R.

El objetivo principal es comparar diferentes modelos de distribución para evaluar el comportamiento del RCS, analizar la calidad de ajuste y estudiar métricas de riesgo.

## Distribuciones consideradas

El repositorio incorpora distintos enfoques de ajuste, entre ellos:

- Lognormal
- Gamma
- Weibull
- Normal
- Exponencial
- Pareto

Cada distribución es evaluada mediante técnicas estadísticas comparables para analizar desempeño y capacidad de representación.

## Contenido de los análisis

Los modelos incluyen:

- Estimación de parámetros mediante Máxima Verosimilitud (MLE)
- Pruebas de bondad de ajuste
- Comparación empírica vs teórica
- Visualización de densidades
- Cálculo de VaR y TVaR
- Simulación
- Métricas:
  - Log-Likelihood
  - AIC
  - BIC
- Exportación de resultados

## Nota sobre confidencialidad

Los datos utilizados en este repositorio fueron anonimizados, modificados y parcialmente simulados con fines exclusivamente ilustrativos.

Las cifras originales asociadas al Requerimiento de Capital de Solvencia (RCS) contienen información de carácter confidencial y de uso interno, por lo que no pueden ser divulgadas públicamente.

Por motivos de privacidad, cumplimiento y confidencialidad institucional, los valores presentados no corresponden a información real ni representan resultados oficiales.

No obstante, la estructura metodológica, el flujo de análisis, las técnicas estadísticas y los procedimientos de modelación utilizados en el repositorio conservan consistencia con aplicaciones reales de análisis actuarial y solvencia.
